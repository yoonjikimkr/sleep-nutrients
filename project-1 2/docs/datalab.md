# 네이버 데이터랩 서비스 소개

네이버 데이터랩 API는 네이버 통합검색 및 쇼핑에서 발생하는 검색 및 클릭 데이터를 조회할 수 있는 서비스입니다.

## 1. 통합검색어 트렌드
네이버 통합검색에서 발생하는 검색어를 연령별, 성별, 기기별(PC, 모바일)로 세분화해서 조회할 수 있는 API입니다.

- **분석 주제군 설정**: 주제어와 하위 주제어(쉼표 구분)를 설정하여 검색 추이를 합산 조회 가능.
- **세분화 정보**: 일간, 주간, 월간 단위 조회 및 5살 단위 연령, 성별, 기기별 구분 가능.
- **상대적 값**: 요청 기간 중 검색 횟수가 가장 높은 시점을 100으로 둔 상대적 값 제공. 절댓값은 제공하지 않음.

## 2. 쇼핑인사이트
네이버 통합검색의 쇼핑 영역과 네이버쇼핑 서비스의 검색 결과에서 사용자가 클릭한 데이터를 조회할 수 있는 API입니다.

- **분야별 통계**: 4단계 카테고리를 통한 세부 통계 정보 제공.
- **검색어 클릭 추이**: 지정 분야의 상품 검색 클릭 추이 파악 (재고 관리 등에 활용 가능).
- **구매층 분석**: 연령별, 성별, 기기별 클릭 비중 및 추이 파악.
- **검색어 통계**: 상품 관련 검색어의 검색 추이와 세부 통계 제공.
