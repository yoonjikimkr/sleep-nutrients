# 뉴스 기사 기반 수면 이슈 분석 대시보드 작업 가이드

본 문서는 `project-1/articles/` 폴더에 정제된 뉴스 기사 CSV 데이터를 활용하여 Streamlit 대시보드를 구축하기 위한 작업 가이드입니다.

## 1. 기본 환경 및 도구
- **Framework**: Streamlit
- **Visualization**: Plotly (Express & Graph Objects)
- **Data Handling**: Pandas
- **Language**: Python (한글 폰트 설정: `koreanize-matplotlib` 또는 Plotly 기본 한글 지원 활용)

## 2. 데이터 소스 및 전처리
- **위치**: `project-1/articles/`
- **대상 파일**:
  - `articles_불면_증가.csv`
  - `articles_멜라토닌_부작용.csv`
  - `articles_수면제_의존.csv`
- **전처리 사항**:
  - `pubDate` 컬럼을 `datetime` 객체로 변환하여 월별/년별 그룹화 준비.
  - `title`과 `description` 텍스트를 결합하여 키워드 매칭 분석 수행.

## 3. 시각화 상세 요구사항

### 3.1. [Line Chart] 월별 기사 발행 트렌드
- **목적**: 시간에 따른 수면 관련 보도량 변화 분석
- **내용**: **"불면 증가"**와 **"멜라토닌 부작용"** 두 키워드의 월별 기사 수 추이를 하나의 그래프에 시각화.
- **차트 타입**: multi-line chart (Plotly)

### 3.2. [Bar Chart] 멜라토닌 부정 이슈 비율 분석
- **대상**: `articles_멜라토닌_부작용.csv`
- **부정 키워드 리스트**: `["부작용", "의존", "내성", "중독", "위험", "경고", "과다복용", "우울", "두통"]`
- **로직**:
  1. 각 기사별로 제목+내용에 위 키워드 중 하나 이상이 포함되었는지 판단.
  2. 년도별로 (부정 키워드 포함 기사 수 / 전체 기사 수) 비율 계산.
- **차트 타입**: 년도별 막대 그래프 (Annual Negative Sentiment Ratio)

### 3.3. [Area/Line Chart] 대체 성분 및 비호르몬 키워드 트렌드
- **대상**: `articles_멜라토닌_부작용.csv`
- **분석 키워드 리스트**: `["마그네슘", "테아닌", "GABA", "감태", "락티움", "비호르몬", "자연유래", "대체"]`
- **로직**: 년도별로 각 성분 키워드가 언급된 빈도(기사 수)의 추이를 계산.
- **차트 타입**: 년도별 언급 빈도 추이 시각화 (Stacked Area Chart 또는 Line Chart)

## 4. 대시보드 레이아웃 구성
- **Header**: 수면 이슈 뉴스 데이터 분석 리포트
- **Sidebar**: 분석 기간 필터, 키워드 선택 필터 등 (필요 시)
- **Main View**:
  - Tab 1: 보도 트렌드 (3.1 요구사항)
  - Tab 2: 멜라토닌 부작용 심층 분석 (3.2 요구사항)
  - Tab 3: 대체 시장 트렌드 (3.3 요구사항)
  - Tab 4: 데이터 요약 테이블 (정제된 원본 데이터 뷰어)
