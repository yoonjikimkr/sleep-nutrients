# Project 1: 네이버 통합 검색어 트렌드 분석

네이버 광고/데이터랩 API를 활용하여 특정 키워드의 검색 트렌드를 수집하고 시각화하는 프로젝트입니다.

## 폴더 구조
- `data/`: 수집된 JSON/CSV 원본 데이터 및 가공 데이터 저장
- `images/`: 분석 결과 그래프 및 차트 이미지 저장
- `instruction_search_trend.md`: 상세 작업 지침서

## 실행 방법
1. `.env` 파일에 네이버 API Client ID/Secret 설정
2. `instruction_search_trend.md`의 안내를 따라 수집 및 분석 코드 작성 및 실행
