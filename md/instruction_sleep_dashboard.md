# 수면 데이터 분석 Streamlit 대시보드 제작 작업지시서

이 문서는 수집된 수면 관련 키워드("불면", "잠이 안 와", "수면 영양제", "멜라토닌") 데이터를 활용하여 시각화 및 인사이트를 제공하는 **Streamlit 대시보드**를 제작하기 위한 지침입니다.

## 1. 개요
- **목표**: 수면 관련 트렌드 및 블로그/쇼핑 데이터를 한눈에 확인하고 키워드별로 비교 분석
- **도구**: Streamlit, Pandas, Plotly (주요 시각화 도구)
- **데이터 소스**: `project-1/data/` 내부의 JSON 및 CSV 파일

## 2. 디자인 및 구조 원칙
1.  **Rich Aesthetics**: 다크 모드 또는 세련된 컬러 팔레트를 사용하고, Plotly의 대화형 그래프를 활용하여 프리미엄한 디자인을 구현합니다.
2.  **Interactive Filters**: 사이드바를 통해 분석 대상 키워드를 선택하거나 기간을 조절할 수 있게 합니다.
3.  **No Placeholders**: 실제 수집된 데이터를 바탕으로 의미 있는 지표(Mean, Max, Sum 등)를 노출합니다.

## 3. 대시보드 메뉴 및 기능 구성 (Tab 기준)

### Tab 1: 📊 트렌드 비교 분석 (5년 시계열)
- **기능**: 사용자가 선택한 키워드간의 5년치 검색 트렌드 비교
- **구현 내용**:
    - **[Chart 1]**: Plotly Line Chart - 일자별 통합 검색량 추이 (다중 키워드 시뮬레이션)
    - **[Table 1]**: 키워드별 최소/최대/평균 검색량 요약 표

### Tab 2: ✍️ 블로그 인사이트 (콘텐츠 분석)
- **기능**: 블로그 게시물 제목 및 링크를 통한 여론 확인
- **구현 내용**:
    - **[Chart 2]**: Plotly Bar Chart - 제목에서 가장 많이 언급된 단어 Top 10
    - **[Table 2]**: 최신 블로그 포스트 리스트 (제목, 날짜, 링크 포함)

### Tab 3: 🛒 쇼핑 트렌드 및 상품
- **기능**: 수면 영양제 등 커머스 데이터 분석
- **구현 내용**:
    - **[Chart 3]**: Plotly Box Plot - 가격대 분포 (쇼핑 데이터 기준)
    - **[Chart 4]**: Plotly Pie Chart - 인기 브랜드 또는 판매처 비중
    - **[Table 3]**: 최저가/최고가 상품 Top 5 비교표

### Tab 4: 🔍 심층 EDA (기초 통계)
- **기능**: 수집된 전체 데이터의 기술 통계 확인
- **구현 내용**:
    - **[Chart 5]**: Plotly Scatter Plot - 검색량과 블로그 발행량 간의 상관관계 (가공 필요 시)
    - **[Table 4]**: 데이터 전체 현황 요약 (행 수, 컬럼 수, 결측치 등)
    - **[Table 5]**: 일자별 최고 검색량 발생 시점의 블로그 글 매칭 리스트

## 4. 핵심 기술 요구사항
- **시각화 (Plotly)**: 최소 5가지 이상의 그래프 (Line, Bar, Pie, Box, Scatter 등)
- **데이터 테이블 (Streamlit)**: 최소 5가지 이상의 표 (`st.dataframe` 또는 `st.table` 활용)
- **키워드 검색**: 사이드바의 `st.multiselect`를 통해 키워드를 동적으로 변경하여 데이터를 필터링합니다.

## 5. 실행 가이드
1.  **환경 구성**: `pyproject.toml`에 명시된 `streamlit`, `plotly` 설치 확인
2.  **스크립트 작성**: `project-1/app.py` 파일 생성
3.  **실행 명령**:
    ```bash
    streamlit run app.py
    ```
