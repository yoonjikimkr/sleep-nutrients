# 네이버 오픈API 비로그인 방식 가이드

비로그인 방식 오픈 API는 클라이언트 아이디와 시크릿만으로 호출 가능한 API입니다.

## 1. 개요
별도의 로그인 인증(OAuth 토큰) 없이 HTTP 요청 헤더에 인증 정보를 포함하여 간편하게 사용 가능합니다.

## 2. 주요 비로그인 방식 API
- **데이터랩**: 검색어 트렌드, 쇼핑인사이트 실행.
- **검색**: 뉴스, 백과사전, 블로그, 쇼핑, 웹문서 등 검색 결과 반환.
- **이미지/음성 캡차**: 자동입력 방지 기능 제공.
- **네이버 공유하기**: 콘텐츠를 네이버 서비스에 공유.
- **Clova Face Recognition**: 얼굴 인식 및 얼굴 감지 기능.

## 3. 인증 요청 헤더
```http
X-Naver-Client-Id: {애플리케이션 클라이언트 아이디}
X-Naver-Client-Secret: {애플리케이션 클라이언트 시크릿}
```
참고: [API 공통 가이드](https://developers.naver.com/docs/common/openapiguide/)
