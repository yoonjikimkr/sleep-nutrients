# 🌙 Sleep Nutrients Project

글로벌(iHerb) 및 국내(Naver) 데이터를 통합 분석하여, 최적의 수면 영양제 복합제 기획과 시장 전략을 수립하는 프로젝트입니다.

---

## 🎯 3040 직장인 타겟 수면 전략 EDA
3040 직장인을 위한 이른바 **'스위치 오프(Switch-OFF)'**를 위한 핵심 성분과 마케팅 로직입니다.

### 💡 3040 맞춤형 성분 밸류맵

| 성분명 (ENG) | 한글 매칭명 | 전략적 가치 | 핵심 근거 및 추천 활용 |
| :--- | :--- | :---: | :--- |
| **L-THEANINE** | 테아닌 | **10.00** | **스트레스 완화**: 알파파 유도로 퇴근 후 긴장된 뇌를 식히기에 최적 |
| **MAGNESIUM GLYCINATE** | 킬레이트 마그네슘 | **9.04** | **고흡수/이완**: 근육 긴장 완화 및 심신 안정에 탁월 (3040 베이스) |
| **ASHWAGANDHA** | 아슈와간다 | **9.00** | **코르티솔 케어**: 스트레스 호르몬 직접 조절로 번아웃 방지 |
| **GABA** | 가바 | **8.00** | **뇌 각성 진정**: 퇴근 후에도 사라지지 않는 업무 잔상을 억제 |
| **VITAMIN B6** | 비타민 B6 | **8.00** | **호르몬 합성**: 수면-각성 주기에 필수적인 멜라토닌 합성 유도 |
| **VALERIAN ROOT** | 발레리안 | **7.06** | **천연 스위치**: 강력한 입면 효과가 필요할 때 보조 성분으로 추천 |
| **LEMON BALM** | 레몬밤 | **7.00** | **불안 해소**: 업무적 압박감을 느끼는 타겟을 위한 릴렉스 성분 |
| **5-HTP** | 5-HTP | **6.00** | **무드 조절**: 전반적 정서 개선을 통한 수면의 질 보강 |
