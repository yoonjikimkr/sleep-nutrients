네이버 검색 API 기반 수면 관련 건강 고민 및 성분 트렌드 분석 EDA 기획안
1. 프로젝트 목적
네이버 검색 API를 활용하여:
사람들이 “수면”과 관련해 가장 많이 검색하는 건강 고민은 무엇인지 파악하고
수면과 함께 자주 검색되는 성분은 무엇인지 분석하며
이 모든 검색 패턴이 성별 및 연령대별로 어떻게 다른지 정량적으로 비교한다.
이를 통해 수면 건강 시장의 수요 구조, 세그먼트별 관심사 차이, 성분 트렌드 방향성을 도출한다.
2. 핵심 분석 질문 (Key Questions)
2.1 건강 고민 관련
“수면”과 함께 가장 많이 검색되는 고민 키워드는 무엇인가?
예: 불면증, 수면장애, 깊은 잠, 뒤척임, 야간 각성, 코골이, 수면무호흡 등
증상 중심 검색이 많은가, 해결책 중심 검색이 많은가?
예: “잠이 안 와요” vs “수면 영양제 추천”
2.2 성분 관련
수면과 함께 가장 자주 검색되는 성분은 무엇인가?
예: 멜라토닌, L-테아닌, 마그네슘, GABA, 감태추출물 등
성분 단독 검색이 많은가, 증상+성분 결합 검색이 많은가?
예: “불면증 멜라토닌”, “깊은잠 마그네슘”
2.3 성별/연령대 차이
연령대별로 검색하는 고민이 어떻게 다른가?
20대: 스트레스/시험/수면패턴 붕괴?
40~50대: 갱년기, 야간 각성?
성별에 따라 성분 선호가 달라지는가?
특정 연령대에서만 강하게 나타나는 성분 트렌드는 무엇인가?

5. 전처리 및 분류 전략
5.1 키워드 유형 분류
키워드를 다음과 같이 분류:
증상형
불면증, 수면장애, 새벽에 깨요 등
감정/상태형
스트레스, 예민함, 긴장
해결책형
수면 영양제 추천, 수면에 좋은 음식
성분형
멜라토닌, 마그네슘, L-테아닌 등
Rule-based 또는 키워드 사전 기반 분류.
5.2 성분 사전 정의

6. 분석 구조 (EDA 설계)
6.1 전체 검색 구조 분석
전체 키워드 중 유형별 비중
상위 50/100 키워드 랭킹
고민 유형별 검색량 합계
시각화 예시:
유형별 비중 바차트
워드클라우드
상위 키워드 Top 20 테이블
6.2 성분 검색 트렌드 분석
성분별 검색량 합계
성분 단독 검색 vs 증상+성분 결합 검색 비율
성분 간 동시 등장 패턴 분석
네트워크 그래프:
노드: 성분/증상
엣지: 함께 등장 빈도
6.3 성별 차이 분석
키워드별 남성/여성 비중 차이
여성 비중이 높은 키워드 Top N
남성 비중이 높은 키워드 Top N
예시 인사이트 방향:
여성: 갱년기, 호르몬, 감태?
남성: 코골이, 수면무호흡?
6.4 연령대별 차이 분석
연령대별:
고민 유형 비중 비교
성분 선호 차이
검색 집중 키워드 비교
히트맵:
X축: 연령대
Y축: 키워드 또는 성분
값: 검색 비중
7. 세그먼트별 페르소나 도출
데이터 기반으로 다음과 같은 페르소나 도출 가능:
20대 여성: 스트레스성 수면 문제 + 자연 성분 관심
40대 여성: 갱년기 + 멜라토닌/감태 관심
30~40대 남성: 코골이/수면무호흡 중심
50대 이상: 만성 불면 + 병원/의학적 접근
8. 확장 분석 아이디어
8.1 시간대별 분석 (가능한 경우)
시험 기간/연말/환절기 등 특정 시점에서 검색량 급증 여부
8.2 고민 → 성분 매핑
특정 고민에서 가장 많이 연결되는 성분은 무엇인가?
예: “갱년기 잠” → 멜라토닌?
“스트레스 잠” → L-테아닌?
8.3 해결책 단계 분석
검색 단계를 Funnel로 해석:
문제 인지 (잠이 안 와요)
정보 탐색 (불면증 원인)
해결책 탐색 (수면 영양제 추천)
특정 성분 탐색 (멜라토닌 3mg)
9. 기대 인사이트
수면 문제의 주된 검색 동기는 무엇인가?
멜라토닌이 실제 검색 시장에서도 중심 성분인가?
특정 세그먼트에서 블루오션 성분은 무엇인가?
브랜드 메시지 및 제품 기획 시 어떤 고민을 전면에 내세워야 하는가?
10. 한계 및 유의사항
검색량은 실제 구매 의도와 완전히 일치하지 않음
네이버 사용자 특성이 전체 인구를 대표하지 않을 수 있음
API 제공 데이터 범위 제한 가능성 존재
동일 키워드의 중복 의도(예: 코골이 치료 vs 정보 탐색) 구분 필요
